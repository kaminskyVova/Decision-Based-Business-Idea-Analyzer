
# Коды отказов и возвратов (единый справочник) 

Gatekeeper использует единый набор reason codes для фиксации причин отказа или возврата.

| Код | Описание |
|----|---------|
| RC-01 | Недостаточный стартовый капитал |
| RC-02 | Неучтённые обязательные расходы |
| RC-03 | Отсутствие принятия ответственности |
| RC-04 | Несоответствие реальности |
| RC-05 | Нарушение законности |
| RC-06 | Попытка продавить систему |
| RC-07 | Перекладывание ответственности / «реши за меня» |
| RC-08 | Запрос вне рамок задачи |
| RC-09 | Несоответствие ресурсным возможностям |
